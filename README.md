# Loja Virtual Nike

Projeto de **Loja Virtual** desenvolvido em **HTML, CSS e JavaScript**.  
O site simula uma loja da Nike, com menu fixo, lançamentos, destaques, seções masculinas, femininas e infantis, além de barra de busca e interação com produtos.

---

## 🔹 Funcionalidades

- Menu fixo com navegação por setores (Masculino, Feminino, Infantil, Destaques, Lançamentos)
- Hover nos menus com dropdown dinâmico usando JavaScript
- Barra de busca simulada
- Seções de destaque com imagens e textos interativos
- Layout responsivo para desktops
- Botões com efeitos hover e animações simples
- Grid de imagens para produtos

---

## 🔹 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (DOM manipulation)
